# npm-why [![npm-version][npm-badge]][npm-link]

Identifies why a package has been installed.

![npm-why-screenshot][screenshot]

## Install & Usage

```bash
npm install -g npm-why
```

```bash
Usage

  $ npm-why <package-name>

Examples

  $ npm-why babel-core
```
