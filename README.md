# npm-why [![npm-version][npm-badge]][npm-link]

## Install & Usage

```bash
npm install -g npm-why
```

```bash
  npm-why - Identifies why a package has been installed.

  Usage

    $ npm-why <package>

  Examples

    $ npm-why babel-core
```
